Window Picker

What is it?

A Mootools plugin that facilitates visually sorting through a stack of dom elements.

Please note that this idea uses (at the time of this writing) some progressive technology, and consequently only works on 'progressive' browsers. I've tested this script successfully on Firefox 3.6.2, Chrome and Opera 10.51

Show me.

This page acts as a simple demo. While the webpage has focus, press CTRL-Left, or CTRL-Right on your keyboard. You can also visit here to see another example.

Not working?

  • The web page must have keyboard focus, not things like the address bar. (Left click somewhere in the page)
  • You need a browser that supports this effect. (Do yourself a favour and make sure you're using FF, Chrome or Opera)
